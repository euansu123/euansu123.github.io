C++指针详解

发表于 2020-05-28 更新于 2020-05-29 Valine：

指针是一个包含了另一个变量地址的变量，即指针也就是指向另一个变量地址的变量。指针指向的变量可以是任意对象，例如变量、结构体、指针等。

指针运算符

C++指针提供了两种运算符，一种是取地址运算符&，一种是间址运算符*。

  • 取地址运算符&，&是一元运算符，返回操作数的内存地址。例如，如果var是一个整型变量，则&var是这个整型变量的地址。&var读作“var的地址”。特别注意：&不能对没有地址的东西取运算符。例如：&(a+b)、&(a++)、&(++a)。

  • 间址运算符*， *是一元运算符，返回操作数所指定地址的变量的值。

一些特殊的指针

  • NULL指针

    在变量声明的时候，如果没有确切的地址可以赋值，为变量赋值为NULL，也就是空指针。

    在大多数的操作系统上，程序不允许访问地址为 0 的内存，因为该内存是操作系统保留的。然而，内存地址 0 有特别重要的意义，它表明该指针不指向一个可访问的内存位置。

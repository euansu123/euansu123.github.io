numpy.array的合并

array([[1, 2, 3, 1, 2, 3], [4, 5, 6, 4, 5, 6]])

这里，我试图让二维矩阵A和一维数组z合并，但它提醒不同维度是不能合并的，但A和z有着相同的列数，而多维的合并也是在多为数组有相同列数的基础上进行的。

所以在这里，我让一维数组变为二维数组，但并不给它添加多余的值。

接下来，就可以完成和二维数组A的合并。

array([[ 1, 2, 3], [ 4, 5, 6], [666, 666, 666]])

多维数组的分割

  • 按照行切割，numpy提供了专门的方法np.vsplit(A, [2])

array([[0, 1, 2, 3], [4, 5, 6, 7]])

  • 按照列切割，numpy提供了专门的方法：np.hsplit(A, [2])

array([[ 0, 1], [ 4, 5], [ 8, 9], [12, 13]])

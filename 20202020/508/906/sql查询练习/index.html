练习题目：

设有一数据库，包含四个表：学生表(Student)、课程表(Course)、成绩表(Score)以及教师表(Teacher)，用SQL语句创建四个表并完成相关题目。

学生表(Student)

属性名 数据类型 可否为空 含义
Sno varchar(20) 否 学号（主码）
Sname varchar(20) 否 学生姓名
Sex varchar(20) 否 学生性别
Sbirthday datetime 可 学生出生年月
Class varchar(20) 可 学生所在班级

课程表(Course)

属性名 数据类型 可否为空 含义
Cno varchar(20) 否 课程号（主码）
Cname varchar(20) 否 课程名称
Tno varchar(20) 否 教工编号（外码）

成绩表(Score)

属性名 数据类型 可否为空 含义
Sno varchar(20) 否 学号（外码，主码）
Cno varchar(20) 否 课程号（外码，主码）
Degree varchar(20) 可 成绩

教师表(Teacher)

属性名 数据类型 可否为空 含义
Tno varchar(20) 否 教工编号（主码）
Tname varchar(20) 否 教工姓名
Tsex varchar(20) 否 教工性别
Tbirthday datetime 可 教工出生年月
Prof varchar(20) 可 职称
Depart varchar(20) 否 教工所在部门

1.查询Student表中的所有记录的Sname、Ssex和Class列。

2.查询教师所有的单位即不重复的Depart列。

3.查询Score表中成绩在60到80之间的所有记录。

4.查询Score表中成绩为85，86或88的记录。

5.以Class降序查询Student表的所有记录。

6.查询Score表中的最高分的学生学号和课程号。（子查询或者排序）

7.查询每门课的平均成绩。

8.查询所有学生的Sname、Cname和Degree列。

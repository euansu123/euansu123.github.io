JQuery学习笔记

Jquery简介

RIA技术

JQuery是RIA技术(Rich Internet Applications)富互联网应具有高度互动性、丰富用户体验以及功能强的客户端。

  • 常见的RIA技术有：Ajax、Flex、Sliverlight、Jquery、Ext。
  • Jquery的优势：轻量级、功能强大的选择q器、出色的DOM封装、可靠的事件处理机制、出色的浏览器兼容性。

Jquery的使用

  • 下载JQuery文件：http://www.Jquery.com，Jquey官网提供了最新的JQuery框架下载，通常只需要下载最小的JQuery包即可。

  • 引用：

    将Jquery框架文件导入后，就可以使用Jquery选择器和各种函数功能。

jquery程序实例

javascrip用querySelector或querySelectorAll作为选择器，jquery用$作为选择器。

jquery选择器

获取html标签后控制输出以及value

jQuery选择器应用

jquery选择不同标签

层次选择器

返回均是集合元素

选择器 描述 示例
$(“ancestor descendant”) 选取ancenstor元素中所有descendant（后代）元素，不仅仅是直接子元素 $(‘body p’).css(‘color’,’red’)，选择body标签中所有的p标签
$(“parent>child”) 选取parent元素下的child（子）元素，直接子元素，例如：不包括子元素中的子元素 $(‘div>p’).css(‘color’,’blue’)，选取div标签下的p标签
$(“prev+next”) 选取紧接在prev元素后的next元素 $(‘div+p’).css(‘color’,’yellow’)，选取紧接在prev元素后的next元素
$(“prev~siblings”) 选取prev元素之后所有siblings元素 $(‘div~p’).css(‘color’,’green’)，选取div标签下的所有p标签
基本过滤选择器
选择器 描述 示例
:first 选取第一个元素 $(‘tr:first’).css(‘background’,’red’)
:last 选取最后一个元素 $(‘tr:last’).css(‘background’,’red’)
:not(selector) 去除所有给定选择匹配的元素 $(‘tr:not(tr:eq(2))’).css(‘background’,’red’)
:even 选择索引是偶数的所有元素，索引从0开始 $(‘tr:even’).css(‘background’,’yellow’)
:odd 选择索引是奇数的所有元素，索引从0开始 $(‘tr:odd’).css(‘background’,’blue’)
:eq(index) 选择索引等于index的元素，索引从0开始 $(‘tr:eq(1)’).css(‘background’,’green’)
:gt(index) 选择索引大于index的元素，索引从0开始 $(‘tr:gt(2)’).css(‘background’,’greenyellow’)
:lt(index) 选择索引小于index的元素，索引从0开始 $(‘tr:lt(3)’).css(‘background’,’orange’)
:header 选择所有的标题元素，如：h1、h2等 $(‘:header’).css(‘background’,’blue’)

内容过滤选择器

内容过滤选择器的过滤规则主要体现在它所包含的子元素和文本内容上。

选择器 描述 示例
:contains(text) 选取含有文本内容为text的元素 $(‘p:contains(内容)’).css(‘color’,’red’)
:empty 选取不包含子元素或者文本的空元素 $(‘p:empty’).text(‘1234’)
:has(selector) 选择含有选择器所匹配的元素 $(‘div:has(p)’).css(‘background’,’red’)
:parent 选取含有子元素或者文本的元素 $(‘:parent’).css(‘background’,’blue’)

可见选择器

可见性选择器是根据元素的可见和不可见状态来选择相应的元素。

选择器 描述 示例
:hadden 选择所有不可见元素 $(‘div:hidden’).css(‘display’,’block’)
:visible 选取所有可见的元素 $(‘div:visible’).css(‘color’,’red’)

属性选择器
选择器 描述 示例
[attribute] 选择拥有此属性的元素 $(‘input[name]’).css({‘width’:’200px’,’height’:’200px’})
[attribute=value] 选取指定属性值为value的元素 $(‘input[name=uname]’).css({‘width’:’200px’,’height’:’200px’})
[attribute!=value] 选取指定属性值不等于value的元素 $(‘input[name!=uname]’).css({‘width’:’200px’,’height’:’200px’})
[attribute^=value] 选取指定属性值以value开始的元素 $(‘input[name^=p]’).css({‘width’:’200px’,’height’:’200px’})
[attribute$=value] 选取指定属性值以value结束的元素
[attribute*=value] 选取指定属性值含有value的元素 $(‘input[name*=e]’).css({‘width’:’200px’,’height’:’200px’})

jquery中的dom操作及事件处理机制
  • 创建和添加
方法 描述 示例
append() 将每个匹配的元素的内部的结尾处追加内容 $(‘body’).append(newP)
appendTo() 将每个匹配的元素追加到指定的元素中的内容 newP.appendTo(‘body’)
prepend() 将每个匹配元素的内部开始处插入内部的开始处 $(‘body’).prepend(newP)
prependTo() 将每个匹配元素插入到指定的元素内部的开始处 newP.prependTo(‘body’)
after() 向每个匹配的元素之后插入内容 $(‘div’).after(newP)
insertAfter() 向每个匹配的元素插入到指定的元素之后 newP.insertAfter(‘div’)
before() 向每个匹配的元素之前插入内容 $(‘div’).before(newP)
insertBefore() 向每个匹配的元素的插入到指定的元素之前 newP.insertBefore(‘div’)

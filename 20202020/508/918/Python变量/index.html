可变和不可变类型

  • 不可变类型，内存中的数据不允许被修改：

    • 数字类型 int，bool，float，complex，long(2,x)
    • 字符串 str
    • 元组 tuple

  • 可变类型，内存中的数据可以被修改：

    • 列表 list
    • 字典 dict
    • 自定义类型 class定义的类型

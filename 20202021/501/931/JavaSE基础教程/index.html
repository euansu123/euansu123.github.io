注释

Java中的注释有三种：单行注释、多行注释、文档注释

  • 单行注释：//
  • 多行注释：/**/
  • 文档注释：/** */

标识符

在命名标识符的时候，我们要特别注意不能与以下关键字相冲突。

Java标识符命名规则及注意点：

  • 所有的标识符都应该以字母（A-Z或a-z）、美元符、或者下划线开始。
  • 首字符之后可以是字母（A-Z或a-z）、美元符、下划线或数字的任何字符组合。
  • 不能使用关键字作为变量名或方法名。
  • 标识符是大小写敏感的。
  • 可以使用中文命名标识符，但不建议。

合法标识符举例：age、Age、$salary、_value

JavaDoc文档

Javadoc命令是用来生成自己的API文档。

JavaDoc命令的参数信息：

  • @author 作者名
  • @version 版本号
  • @since 指明需要最早使用的jdk版本
  • param 参数名
  • return 返回值情况
  • throws 异常抛出情况

用cmd命令行生成Java文档。

也可以使用IDEA生成Java文档。

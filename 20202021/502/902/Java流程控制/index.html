next和nextLine方法的区别：

  • next():
    1. 一定要读取到有效字符后才可以结束输入。
    2. 对输入有效字符之前遇到的空白，next()方法会自动将其去除掉。
    3. 只有输入有效字符后才将其后面的空白作为分隔符或者结束符。
    4. next()不能得到带有空格的字符串。
  • nextLine():
    1. 以Enter为结束符，也就是说nextLine()方法返回的是输入回车之前的所有字符。
    2. 可以获取空格。

do...while循环

定义语法：

对于while循环语句而言，如果不满足条件，则不能进入循环，而do...while循环，即使不满足循环条件也会执行一次循环。

如上运行结果，由于不满足while循环的条件，所以while循环并未执行。也不满足do...while循环的条件，但do...while预先执行一次以及先执行后判断的特性，使得do...while执行力4次循环。

while循环和do...while循环的区别:

  • while先判断后执行，do...while先执行循环后判断。
  • do...while总是保证循环至少执行一次。

java---class---反编译(IDEA)，IDEA可以直接实现class字节码文件的反编译。

反编译的流程：

  • 首先要在IDEA中找到java项目的编译输出文件路径。
